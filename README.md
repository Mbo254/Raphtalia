### The Hard Method
```js
GET QR
$ apt install git
$ apt install nodejs --fix-missing
$ git clone https://github.com/nexusNw/Julie-Mwol
$ cd Julie-Mwol
$ npm install @adiwajshing/baileys
$ npm install chalk
$ node julie.js
```

```js
$ node julie.js
```
